Our Tech Stack

Full stack javascript is our bread and butter. Typically using React for frontend, Node.JS for backend and React Native for cross-platfor mobile app development. The benefits of using full stack javascript are:

  • Easier to hire team members with a single language
  • The Javascript/Node.js community is extremely active in terms of libraries and support
  • Javascript is a constantly evolving language
  • The dynamic and performant nature of the language allow rapid development cycles

We deploy our services on the cloud using Docker and a cluster management system like Kubernetes. These are the same techniques Google uses to ensure their search engine always works, 24/7. The benefits of using these solutions are:

  • State of the art reliability techniques lower the prevalence of site outages
  • Bugs are found and solved faster with continuous integration
  • Easy to move between AWS, Google Cloud, or your own servers
  • Scaling is a breeze with a distributed microservice architecture
