The Arcanium Academy

The Arcanium Academy is a 1-2 year apprenticeship program for web developers. At the Arcanium Academy you will receive a well rounded education that will deeply prepare you for the software development industry.

You will learn a flexible, in-demand javascript development stack. You're going to be building real life projects as part of a team. And you will have the opportunity to earn money while mastering your skills.

Curriculum

We will run you through a curriculum along these lines:

Modern Frontend Web Development

  • modern HTML/HTML5
  • modern CSS
  • ES7+ Javascript
  • React.js
  • Redux pattern
  • Gulp, Webpack & other build tools

Modern Backend and Infrastructure Development

  • Building APIs with Node.js
  • Microservices vs monoliths
  • Databases - MongoDB, PostgreSQL
  • Cache - Redis, Memcached
  • Specialized tools: ZeroMQ, ETCD
  • Deployment - Docker, Kubernetes, Rancher
  • Cloud platforms - Heroku, AWS, Google Cloud, Digital Ocean
  • Testing and continuous integration

UNIX Philosophy and Command Line

  • Text editing with Vim or Emacs
  • How Linux Works
  • Docker
  • Git
  • Zshell
  • Package managers
  • BASH scripts
  • Makefiles
  • UNIX philosophy
  • Understanding hacking, UNIX, Linux & Open Source

Earn while you Learn

After you've learned the basics, we will do our best to secure a source of funding for you. This might involve finding a local internship or working as a freelancer for our agency. We believe this step is 100% fundamental to your success.

Required Reading

Building your library is essential. From the get-go, we will recommend that you read at least 1 chapter of a code related book per day. Here are some of our favorites:

  • Hackers and painters
  • Data intensive systems
  • Design Patterns - Gang of four
  • Code Complete
  • The Pragmatic Programmer
  • Secrets of the Javascript Ninja

Leaving the Nest

Many of our graduates work for Arcanium.io. Beyond that, we will do our best to reach out to our networks to find you a job. With the skills you will learn, you will have no trouble finding a well paying job. Well paying remote work is very attainable.

Location and Meetups

Arcanium is a remote-first, digital nomad organization. That means we work from everywhere. If you want to learn in person, we can schedule a meetup.

The Academy currently has members from Florida, Connecticut, Seattle, Toronto and Denver. We love the great outdoors and traveling, so we are always planning fun vacations for our members. If you like the great outdoors, travel and coding, you'll fit right in!

(yes we code from ski resorts.)

Cost of the Program

We charge a flat $500/month fee for the program. In order to make this attainable, we do our best to help secure funding for our apprentices once they are ready.
